# Beslissingen maken

Leer hoe u [beslissingen](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioning/create-manage-activities/create-offer-activities.html?lang=nl-NL) maakt voor beslissingsbeheer.
